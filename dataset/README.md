# Dataset Overview

This folder contains captured UI state sequences for 4 tasks across GitHub and Linear.

## 1. GitHub – Create repository

- **Task:** On GitHub, create a new public repository named `softlight-demo-repo`.
- **App:** GitHub
- **Folder:** `github_create_repo/run_01`
- **What it shows:**
  1. New repo form on https://github.com/new
  2. Repository name field filled
  3. Repository successfully created and final repo page

## 2. GitHub – Open Issues tab

- **Task:** On GitHub, open the Issues tab for this repository.
- **Folder:** `github_open_issues_tab/run_01`
- **What it shows:** Repo home → Issues tab open.

## 3. GitHub – Create issue

- **Task:** On GitHub, create a new issue titled `Sample bug report from agent` in this repository.
- **Folder:** `github_create_issue/run_01`
- **What it shows:** New issue form → issue detail page after submission.

## 4. Linear – Create workspace

- **Task:** In Linear, after I am logged in, create a new workspace named `Softlight Agent Demo Workspace` using the workspace switcher.
- **Folder:** `linear_create_workspace/run_01`
- **What it shows:** Existing workspace → workspace switcher → create workspace modal → new workspace main page.
